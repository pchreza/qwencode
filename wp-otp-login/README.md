# افزونه ورود و ثبت‌نام با OTP و IPPanel

## راهنمای کامل نصب و استفاده

### نصب افزونه

1. پوشه `wp-otp-login` را در مسیر `/wp-content/plugins/` آپلود کنید
2. به بخش افزونه‌ها در پیشخوان وردپرس بروید
3. افزونه "WP OTP Login with IPPanel" را فعال کنید

### تنظیمات اولیه

1. به منوی **OTP Login** در پیشخوان وردپرس بروید
2. در بخش **تنظیمات IPPanel**:
   - **API Key**: کلید API خود را از پنل IPPanel دریافت و وارد کنید
   - **Sender ID**: شناسه ارسال‌کننده (شماره پیامک) خود را وارد کنید
   - **Pattern Code** (اختیاری): اگر از الگوی پیامک استفاده می‌کنید، کد الگو را وارد کنید

3. در بخش **تنظیمات OTP**:
   - طول کد OTP (پیش‌فرض: 6 رقم)
   - زمان انقضای کد (پیش‌فرض: 300 ثانیه)
   - حداکثر تعداد تلاش برای تأیید
   - متن پیامک (از `%code%` برای قرار دادن کد استفاده کنید)

4. در بخش **تنظیمات کاربران**:
   - ثبت‌نام خودکار کاربران جدید
   - نقش پیش‌فرض کاربران جدید

### روش‌های استفاده در سایت

#### 1. استفاده از شورت‌کد (Shortcode)

می‌توانید از شورت‌کدهای زیر در هر برگه یا نوشته‌ای استفاده کنید:

**فرم ورود:**
```
[otp_login_form]
```

با تنظیمات دلخواه:
```
[otp_login_form 
    redirect_url="https://example.com/dashboard" 
    title="ورود به حساب کاربری"
    show_title="true"
    phone_placeholder="09xxxxxxxxx"
    send_button_text="ارسال کد تأیید"
    verify_button_text="ورود به سایت"
]
```

**فرم ثبت‌نام:**
```
[otp_register_form]
```

با تنظیمات دلخواه:
```
[otp_register_form 
    redirect_url="https://example.com/profile" 
    title="ثبت‌نام در سایت"
    show_title="true"
    phone_placeholder="09xxxxxxxxx"
    firstname_placeholder="نام"
    lastname_placeholder="نام خانوادگی"
    send_button_text="ارسال کد تأیید"
    verify_button_text="ثبت‌نام"
]
```

**نمایش اطلاعات کاربر:**
```
[otp_user_info]
```

با تنظیمات دلخواه:
```
[otp_user_info 
    show_avatar="true" 
    avatar_size="80"
    logout_url="/logout"
    logged_in_text="خوش آمدید"
    logged_out_text="لطفاً وارد شوید"
]
```

#### 2. استفاده از المنتور (Elementor)

اگر از صفحه‌ساز المنتور استفاده می‌کنید:

1. صفحه مورد نظر را با المنتور ویرایش کنید
2. در پنل ویجت‌ها، دسته‌بندی **OTP Login** را پیدا کنید
3. ویجت‌های زیر را به صفحه اضافه کنید:
   - **OTP Login Form**: فرم ورود با OTP
   - **OTP Register Form**: فرم ثبت‌نام با OTP
   - **OTP Input**: فیلد ورودی OTP
   - **User Info**: نمایش اطلاعات کاربر

4. هر ویجت را طبق نیاز خود سفارشی‌سازی کنید

#### 3. ایجاد برگه ورود/ثبت‌نام اختصاصی

برای ایجاد یک برگه ورود یا ثبت‌نام کامل:

1. یک برگه جدید در وردپرس ایجاد کنید (مثلاً با نام "ورود")
2. شورت‌کد `[otp_login_form]` را در آن قرار دهید
3. برگه را منتشر کنید
4. آدرس این برگه را می‌توانید به عنوان صفحه ورود استفاده کنید

همین کار را برای ثبت‌نام با شورت‌کد `[otp_register_form]` انجام دهید.

### نکات مهم

#### فونت وزیر
برای نمایش صحیح فونت وزیر:
1. فونت‌های وزیر را از [GitHub](https://github.com/rastikerdar/vazir-font) دانلود کنید
2. فایل‌های فونت را در پوشه `assets/fonts/` قرار دهید
3. یا از CDN استفاده کنید و مسیر فونت‌ها را در CSS تغییر دهید

#### امنیت
- کدهای OTP در دیتابیس هش می‌شوند
- محدودیت تعداد درخواست بر اساس IP و شماره موبایل
- جلوگیری از حملات Brute Force
- استفاده از Nonce برای تمام درخواست‌های AJAX

#### API IPPanel
این افزونه از آخرین نسخه API IPPanel استفاده می‌کند:
- endpoint ارسال پیامک: `https://api.ippanel.com/api/v1/sends`
- endpoint بررسی موجودی: `https://api.ippanel.com/api/v1/accounts`
- احراز هویت با هدر: `Authorization: AccessKey YOUR_API_KEY`

### عیب‌یابی

#### مشکل در ارسال پیامک
1. API Key و Sender ID را بررسی کنید
2. موجودی حساب IPPanel را چک کنید
3. لاگ‌های خطا را در پوشه `wp-content/debug.log` بررسی کنید

#### مشکل در نمایش فونت
1. مطمئن شوید فایل‌های فونت در پوشه `assets/fonts/` وجود دارند
2. کش مرورگر را پاک کنید
3. از صحت مسیر فایل‌های فونت در CSS اطمینان حاصل کنید

#### مشکل در اتصال به IPPanel
1. از فعال بودن API Key اطمینان حاصل کنید
2. بررسی کنید که سرور شما به `api.ippanel.com` دسترسی داشته باشد
3. لاگ‌های PHP را برای مشاهده خطاهای دقیق‌تر بررسی کنید

### گزارش‌گیری و آمار

در پنل مدیریت، به منوی **OTP Login** بروید و از بخش‌های زیر استفاده کنید:

- **تنظیمات عمومی**: مدیریت تنظیمات افزونه
- **لاگ پیامک‌ها**: مشاهده تاریخچه پیامک‌های ارسالی
- **آمار و ارقام**: مشاهده آمار ورودها، ثبت‌نام‌ها و نرخ موفقیت

### پشتیبانی

برای گزارش مشکلات یا درخواست ویژگی‌های جدید، لطفاً با توسعه‌دهنده تماس بگیرید.
