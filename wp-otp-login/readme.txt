افزونه ورود و ثبت‌نام با پیامک و OTP از طریق IPPanel همراه با ویجت‌های المنتور

== Description ==

این افزونه امکان ورود و ثبت‌نام کاربران با استفاده از کد یکبار مصرف (OTP) ارسال شده از طریق پیامک را فراهم می‌کند. این افزونه با سرویس IPPanel یکپارچه شده و دارای ویجت‌های المنتور برای طراحی فرم‌های ورود و ثبت‌نام است.

**ویژگی‌ها:**

* ورود و ثبت‌نام با کد یکبار مصرف (OTP)
* یکپارچگی کامل با API IPPanel
* پشتیبانی از ارسال معمولی و الگویی (Pattern)
* ۴ ویجت المنتور برای طراحی فرم‌ها
* پنل مدیریت پیشرفته با آمار و گزارشات
* امنیت بالا با استفاده از nonce و rate limiting
* پشتیبانی از شماره‌های موبایل ایران
* قابلیت ثبت‌نام خودکار کاربران جدید
* تایمر برای ارسال مجدد کد
* کاملاً ریسپانسیو و سازگار با RTL

**ویجت‌های المنتور:**

1. فرم ورود با OTP
2. فرم ثبت‌نام با OTP
3. فیلد ورودی OTP
4. نمایش اطلاعات کاربر

== Installation ==

1. پوشه `wp-otp-login` را در مسیر `/wp-content/plugins/` آپلود کنید
2. افزونه را از بخش «افزونه‌ها» در وردپرس فعال کنید
3. به منوی «OTP Login» در پیشخوان وردپرس بروید
4. کلید API و شناسه ارسال‌کننده IPPanel خود را وارد کنید
5. ویجت‌ها را در صفحه‌ساز المنتور استفاده کنید

== Frequently Asked Questions ==

= چگونه کلید API IPPanel را دریافت کنم؟ =

به پنل کاربری خود در ippanel.com وارد شوید و از بخش تنظیمات، کلید API خود را دریافت کنید.

= آیا می‌توانم متن پیامک را تغییر دهم؟ =

بله، در بخش تنظیمات افزونه می‌توانید قالب پیامک را سفارشی‌سازی کنید. از %code% به عنوان متغیر کد استفاده کنید.

= آیا این افزونه با کش کار می‌کند؟ =

بله، اما بهتر است صفحات حاوی فرم ورود را از کش مستثنی کنید.
